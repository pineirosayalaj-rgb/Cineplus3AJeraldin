# Cineplus3AJeraldin
Aplicación básica de cobro para un cine creada en Python con wxGlade. Consta de 4 ventanas: Inicio de Sesión ,taquilla ,dulcería y ticket.

Ventana de Inicio de Sesion (Login)
--El sistema debe de tener una pantalla de inicio de sesion.
--Para este proyecto , en el inicio de sesion se hara con datos preestablecidos en el codigo los cuales seran (Jeraldine como usuario y la contraseña seria: 123).Sino estan esos datos no podras ingresar a las demas ventanas.
--Y para eso debes de ingresar tus datos de usuario y tu contraseña para poder ingresar a la siguiente ventana.

Ventana de Boletos (Taquilla)
--Se crea la pantalla de la ventana de Boletos
--El sistema debe de permitir al empleado seleccionar la pelicula, el horario y la sala.
"Mufasa:El rey león" 14:00, 17:00,20:00 Sala 1
"Las Guerreras k-pop" 13:00, 16:00,19:30 Sala 2
"Megan 2.0" 15:00, 18:00, 21:00 Sala 3

--Se debe de especificar la cantidad de boletos.
-- se debe de indicar si los boletos son para adultos o para menores de edad ,ya que el precio es diferente (adultos $80,menores de edad $80).
--Al finalizar la seleccion de los boletos ,el sistema debe de mostrar un resumen de los boletos adquiridos y calcular el subtotal de la taquilla.

Venta de Productos (Dulceria)
--Despues  de la taquilla ,el sistema debe de preguntar si el cliente desea comprar los productos de la dulceria.
--Si la respuesta es afirmativa, el sistema debe de llevar al empleado a una nueva pantalla.
--Se debe de mostrar 3 combos preestablecidos y cada uno de los precios son diferentes (Combo basico $75, Combo Cuates $130, Combo Familiar $200).
--Por cada combo ,el sistema debe de mostrar su precio, el contenido y una imagen alusiva.
--Tambien se debe de permitir la venta de productos individuales:(Las Palomitas:Tamaños chico $25, mediano $45 y grande $60)(Los refrescos:Sabores Coca cola,Sprit,Manzanita , en tamaño chico $25, mediano $40 y grande $60).
--El empleado debe de poder indicar la cantidad de cada combo o el producto individual.

Resumen de la compra y el cobro (Ticket)
--Al finalizar la seleccion del producto de la dulceria, el sistema debe de mostrar una ventana de resumen.
--Esta ventana debe de incluir un desglose de la compra:
--Seccion de "Taquilla" con los boletos comprados y su subtotal.
--Seccion de "Dulceria"con los combos y los productos individuales y su subtotal.
--Y finalmente , el sistema debe de mostrar el costo total de toda la compra (Taquilla + Dulceria)
